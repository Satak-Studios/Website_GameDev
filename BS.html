BALL SURFERS

WHAT IS BALL SURFERS?
BALL SURFERS is crossplatform platformer multiplayer and singleplayer game in which you have to avoid obsticle by moving left or right and you can also compare your highscore with other players. In multiplayer you have an unlimited obsticle map but with an ending which is impossible to complete
System Requirements
Android
  • RAM = 1GB+ RAM
  • Storage Required = 100mb
  • CPU = ARMv7 with Neon Support (32-bit) or ARM64
  • Graphics API = OpenGL ES 2.0+, OpenGL ES 3.0+, Vulkan
  • Operating system = Android 4.4 (API 19)+
IOS
  • Ram = 1GB+
  • Storage Required = 100mb
  • CPU = A7 SoC+
  • Graphics API = Metal
  • Operating System = IOS 11+
Windows
  • RAM = 2GB+ RAM
  • Storage Required = 100mb
  • CPU = Intel i3
  • Graphics Card = Any Graphics Card is OK
  • Operating system = Windows 10 (x86) or Windows 10 (x64)
